Ambrose＊Starlit 管理画面

GitHub APIを使って投稿を自動でGitHub Pagesに公開します。

🔧 初回セットアップ

1. GitHubリポジトリの準備

• GitHub Pagesが有効なリポジトリを用意
• このアプリのファイルをアップロード

2. Personal Access Token取得

• GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
• "Generate new token (classic)"をクリック
• "repo"にチェックを入れる
• トークンをコピー

3. 設定画面で入力

• リポジトリ名: "username/repository"
• ブランチ名: "main"または"gh-pages"
• Personal Access Token: 取得したトークン

✏️ 投稿方法

画像ボタン

写真や画像を追加できます。複数枚同時に選択可能です。

投稿ボタン

テキストや画像を投稿します。自動的にGitHubにpushされます。

📝 投稿の操作

コピーボタン

投稿のテキストをクリップボードにコピーします。

画像保存ボタン

投稿の画像をすべてダウンロードします。

削除ボタン

投稿を削除します。GitHubにも反映されます。

🏷️ ハッシュタグ

#タグ名 のように # をつけると、投稿をカテゴリ分けできます。

例: 今日の作業進捗！ #開発 #WPF

🔗 URL自動リンク

https:// で始まるURLを入力すると、自動的にリンクになります。
